# Common Development References

## npm

- init
  - `npm init`
- install
  - `npm install [module_name]`
  - when npm install error is occured (windows)
    - `npm install --global --production windows-build-tools`

## git

- init
  - `git init`
- add
  - `git add [filename]`
  - `git add *`
- commit
  - `git commit -m "message"`
- status
  - `git status`
- remove
  - `git rm [filename]`
- rename
  - `git mv [old_filename] [new_filename]`
- history, log
  - `git history`
  - `git log`
